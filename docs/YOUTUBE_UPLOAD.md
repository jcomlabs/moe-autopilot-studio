## Description

The fastest decode setting is not always the fastest workload.

MoE Autopilot Studio is a Windows-first lab that turns measured MoE routing
evidence into a reproducible ENABLE, DISABLE, or MEASURE verdict. This demo
shows the same split changing from ENABLE to DISABLE as workload shape changes,
the protocol guard that blocks invalid benchmark arithmetic, a bounded GPT-5.6
advisor through Codex App Server and ChatGPT OAuth, and a structured local
runner that never executes shell strings.

work inside a deterministic boundary; it cannot replace metrics, commands,
